# Galicia Badges

> Un proxecto por e para as comunidades galegas

Este é un proxecto creado co obxectivo de fomentar a asistencia recorrente a eventos mediante unha dinámica entretida, cooperativa e baseada en medallas/badges.

## ➕ Como sumar a miña comunidade?

1. **Deseñar unha medalla**  
   Preferentemente cun deseño que distinga a data ou o mes do evento, especialmente no caso de eventos recorrentes.

2. **Contactar cos admins do proxecto**  
   - 📧 Correo: `sysarmygalicia@gmail.com`  
   - 💬 Telegram: `@Qrow01`

3. **Xerar un QR durante os eventos**  
   En cada evento xerarase un código QR dun só uso para cada persoa asistente.
